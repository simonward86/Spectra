@spec1d/plus

This is a the reference documentation for the function @spec1d/plus This function gives a new spectra with a value of spectra s1 plus a value or an addition of values from spectra s1 and s2.

Contents

  • Usage
  • Inputs
  • Outputs
  • Notes
  • Examples
  • Example 1
  • Example 2
  • Example 3

Usage

s_out = plus(s1,s2)
s_out = s1 + s2
s_out = s1 + a
s_out = a + s1

Inputs

  1. r = s1 + s2 , where s1 and s2 are spectra
  2. r = s1 + a , where a is to be added to the y axis
  3. r = s1 + [xc,yc], where xc (yc) is added the x (y) axis

Outputs

spec1d object s_out

Notes

  • If spectra are not the same length, s2 is interpolated to the x values of s1
  • @spec1d/plus should not be used to combine counts taken with different monitors - use combine(s1,s2)

Examples

These are some examples on using @spec1d/plus

Example 1

Addition of spectra and value;

s = spec1d(1:5,1:5,0.1);
s_1 = s + 5;
s_1 = 5 + s;

Example 2

Addition two spectra (equal points)

s2 = spec1d(1:5,2:6,0.15);
s_2 = s + s2;

Example 3

Addition two spectra (unequal points). This using interpolation on s3

s3 = spec1d(1:5 + 0.1,2:6,0.15);
s_3 = s + s3;
