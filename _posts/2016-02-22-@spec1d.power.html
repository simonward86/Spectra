@spec1d/power

This is the reference documentation for the function @spec1d/power

This function raises the y-values of a single or vecor of spec1d objects to a power specified.

Contents

  • Syntax
  • Inputs
  • Outputs
  • Example
  • See Also

Syntax

  s_out = power(s,m)
  s_out = s^m
  s_out = s.^m

Inputs

  • s - Single or vector of spec1d objects.
  • m - Single value corresponding to the power to raise spectra s by.

Outputs

  • s_out - Spec1d object of the same size as the input array

Example

This is an example on using @spec1d/power

Example 1

s = spec1d(1:0.2:5,linspace(1,2,21),0.1);
s_1 = s.^2;

figure
plot(s,s_1)
legend({'s','power(s,2)'},'Location','NorthWest')
