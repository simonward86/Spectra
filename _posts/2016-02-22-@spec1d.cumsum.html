@spec1d/cumsum

This is the reference documentation for the function @spec1d/cumsum

This function gives the cumulative summation of points in each spectra s were s can be single or an array.

Contents

  • Syntax
  • Inputs
  • Outputs
  • Example
  • See Also

Syntax

  s_out = cumsum(s)

Inputs

  • s - Single or vector of spec1d objects

Outputs

  • s_out - Spec1d object of the same size as the input array

Example

This is an example on using @spec1d/cumsum

Example 1

s = spec1d(1:0.2:5,sin(linspace(0,2*pi,21)),0.1);
s_1 = cumsum(s);

figure
plot(s,s_1)
legend({'s','cumsum(s)'})
