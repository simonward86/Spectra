@spec1d/sqrt

This is the reference documentation for the function @spec1d/sqrt

This function takes the square root of a single or vecor of spec1d objects.

Contents

  • Syntax
  • Inputs
  • Outputs
  • Example
  • See Also

Syntax

  s_out = sqrt(s)

Inputs

  • s - Single or vector of spec1d objects

Outputs

  • s_out - Spec1d object of the same size as the input array

Example

This is an example on using @spec1d/sqrt

Example 1

s(1) = spec1d(1:5,1:5,0.1);
s(2) = spec1d(1:0.2:5,sin(linspace(0,2*pi,21)),0.1);
s_1 = sqrt(s(1));

figure
plot(s(1),s_1)
legend({'s','sqrt(s)'},'Location','NorthWest')

Example 2

Immaginary numbers are not supported

s_2 = sqrt(s(2));

Error using spec1d/sqrt (line 14)
Some or all y-values in the input spec1d object are negative. spec1d does not support imaginary numbers.
