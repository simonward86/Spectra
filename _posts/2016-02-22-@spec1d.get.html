@spec1d/get

This is the reference documentation for the function @spec1d/get

This function is an implementation of the get/set routine from the handle class. Using get it is possible to get the fields in a spec1d object.

Contents

  • Syntax
  • Inputs
  • Outputs
  • Examples
  • See Also

Syntax

  m = get(s,'m');
  [m, n] = get(s,'m','n');

Inputs

  • s - Single or vector of spec1d objects

Outputs

  • m - Column vecor of field '_m_'.

Examples

These are examples on using @spec1d/get

Example 1

Get the x-values of a spectra

s = spec1d(1:0.2:5,sin(linspace(0,2*pi,21)),0.1);
x = get(s,'x');
fprintf('x\n')
fprintf('%0.03f\n',x')

Get the x and y values of a spectra

fprintf('%0.03f\t%0.03f\n',[x y]')
