@spec1d/rebin

This is a the reference documentation for the function @spec1d/rebin.

This function re-bins a spec1d spectrum s to the bins specified by dx.

Syntax

s_out = rebin(s,dx,varargin)

Inputs

  • s - Single spec1d object or array
  • dx - Binning argument
    • scalar - Specifies bin widths.
    • vector - Specifies the bin centers.
    • sped1d - Uses x-values of dx as bin centers
    • string - Use a binning algorithm described in the documentation histcounts.
  • varargin - Optional Parameter/Value pairs

Optional Inputs

The optional method controls how the x-value of the bin is interpreted. Options are:

  1. average : New x values are average within bins (weighted by error)
  2. force : New x values are as specified in dx
  3. interp : New x-values as specified by dx, y and e interpolated from 'average'-x,

The default is average

The methods above can also have the byError flag.

  1. 0 : Data is binned by x. 'Default'
  2. 1 : Data is binned to a constant error. Only method, average is supported.

Outputs

  • s_out - spec1d object

Examples

These are some examples on using @spec1d/rebin. Rebin spectra s1.

Example 1

Rebin s to x-value steps 0.5

s = spec1d(1:0.2:5,sin(linspace(0,2*pi,21)),0.1);
s_out1 = rebin(s,0.5);

figure
plot(s,s_out1)

Example 2

Rebin s to an x vector

x = 1:0.4:5;
x(3:4:end) = [];
s_out2 = rebin(s,x);

figure
plot(s,s_out2)

Warning: Supplied bin range does not cover data. Removing points.

Notes

The width of boundary bins is twice the distance to neighbour bin.
